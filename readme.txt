=== Layout Engine Base ===

It is demonstration theme for designers/developers to learn how they can integrate <a href="http://wordpress.org/extend/plugins/layout-engine/">Layout Engine framework</a> features and powerful LessCSS. 
<a href="https://www.youtube.com/watch?v=9L_C5lRCOl0">Watch in action</a>

All hacks are included in "<b>functions.php</b>" to demonstrate
- "compatibility.php" How to display notice on frontend/backend if layout engine is missing.
- "lesscss_options.php" How to manipulate built in LessCSS compiler options.
- "layout.php" define default layout, refine automatic html markup to standard one and learn how to create additional drag and drop blocks.
- "hard_coded_filters.php" demonstrate how you can create blocks programmatically which cannot be changed by users.
- "block_items/*" provide some basic instruction to create your own drag and drop element (block item)

== Installation ==

You can download and install Layout Engine Base theme using the built in WordPress theme installer. If you download manually, make sure it is uploaded to "/wp-content/themes/layout-engine-base/".

Activate Layout Engine Base in the "Appearance -> Themes" admin panel using the "Activate" link.

If Layout Engine is not installed or activated, it will show you notice bar asking to install the framework using WordPress plugin installer or manually.
